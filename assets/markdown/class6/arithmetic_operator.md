## Arithmetic Operators

![](resource:assets/images/C++/img11.png)

```
#include<iostream>
using namespace std;
int main()
{
int a=5,b=3;
cout<<(a+b)<<"\n";
cout<<(a-b)<<"\n";
cout<<(a*b)<<"\n";
cout<<(a/b)<<"\n";
cout<<(a%b)<<"\n"; 	// %(modulus) holds remainder
}

/*
### Output ###
8
2
15
1
2
*/
```
